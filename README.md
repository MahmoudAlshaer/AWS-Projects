#  AWS Cloud Projects Portfolio

A collection of hands-on AWS projects demonstrating 
real-world cloud computing skills including 
networking, security, data visualization, 
and static web hosting.

---

##  Projects

### 1. [AWS Cloud Security with IAM and EC2](./AWS%20Cloud%20Security%20with%20IAM%20and%20EC2)
Implemented a secure AWS environment using IAM 
users and custom JSON policies. Created dev and 
production EC2 environments with restricted access 
control, verified using real permission testing.

**Services:** AWS IAM, Amazon EC2
**Skills:** Security policies, least privilege, 
access control, environment separation

---

### 2. [AWS Data Visualization with QuickSight](./AWS%20Data%20visualization%20with%20QuickSight)
Analyzed a 3.2MB Netflix titles dataset using 
Amazon QuickSight. Built an interactive dashboard 
with multiple charts showing content trends 
across genres, release years, and content types.

**Services:** Amazon QuickSight, Amazon S3
**Skills:** Data visualization, dashboard creation, 
S3 data sources, business intelligence

---

### 3. [AWS S3 Static Website Hosting](./AWS%20S3%20static%20website%20hosting)
Deployed a static website using Amazon S3. 
Configured bucket permissions, enabled static 
hosting, and troubleshot a 403 Forbidden error 
by implementing the correct bucket policy.

**Services:** Amazon S3
**Skills:** Static hosting, bucket policies, 
public access configuration, troubleshooting

---

### 4. [AWS VPC Network Infrastructure](./AWS%VPC%Network)
Built a custom Virtual Private Cloud from scratch 
including subnet configuration and internet 
gateway setup to enable secure network 
communication on AWS.

**Services:** Amazon VPC, Subnets, 
Internet Gateway
**Skills:** Network architecture, CIDR planning, 
cloud networking fundamentals
